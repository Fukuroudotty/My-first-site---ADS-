- Plataforma CIDTS

O que é?

A Plataforma CIDTS é um espaço online dedicado a destacar e compartilhar os projetos inovadores desenvolvidos pelo Centro de Inovação e Difusão de Tecnologias para o Semiárido (CIDTS). Sua missão é oferecer uma visão abrangente dos avanços e iniciativas que visam aprimorar e impulsionar o setor agropecuário.

Objetivos

A Plataforma CIDTS se destina a apresentar os projetos inovadores desenvolvidos pelo Centro de Inovação e Difusão de Tecnologias para o Semiárido (CIDTS). Nesse espaço virtual, você encontrará uma visão aprofundada dos avanços e iniciativas que têm como foco aprimorar e impulsionar o setor agropecuário.

• Comércio eletrônico
Os produtores poderão vender seus produtos e serviços através de uma plataforma simples e de fácil acesso.

• Estação Meteorológica
Realiza a medição de dados meteorológicos para informar os produtores sobre o clima da região.

• Sistema de Análise Economica
Os produtores poderão estimar o valor do animal e de sua carne com base nos gastos.

A Plataforma CIDTS desempenha um papel crucial ao oferecer uma vitrine virtual para os projetos inovadores concebidos pelo Centro de Inovação e Difusão de Tecnologias para o Semiárido.

- Escrituração zootécnica 4.0: Identificação única de pequenos ruminantes com tecnologia de internet das coisas

O que é?

O projeto RuminWeb se baseia na criação de um aplicativo com o intuito de ajudar aos criadores de pequenos ruminantes à coleta e armazenamento de dados sobre acriação.

Objetivos

• Realizar a Identificação única de pequenos ruminantes
• Desenvolver um APP para receber dados das Tags RFID
• Desenvolver um sistema Web para administração Zootécnica

• Identificação por RFID
RFID é uma tecnologia que se usa de tags eletrônicas para identificar. No projeto, será utilizado para rastrear ovinos e caprinos através de ondas de rádio.

• Registro do animal
Este documento será utilizado para identificar e monitorar animais, contendo informações como nome, espécie, data de nascimento e histórico veterinário

• Análise de dados
É esta parte do processo onde irá extrair-se informações relevantes a partir de conjuntos de dados para tomar decisões informadas e obter vantagens.

• Tomada de decisão
O agricultor, com as informações obtidas, toma a decisão do que será feito a seguir com o animal. Se ele será vermifugado, vacinado, vendido e etc.

A escrituração zootécnica é um passo muito importante para a criação de animais. Digitalizá-las tornará essa ação muito mais completa e facilitada. Este é o objetivo do RuminWeb.

- Classificação de predadores de abelhas meliferas utilizando redes neurais convolucionais e transferência de conhecimento

O que é?

O projeto consiste na captação de imagens das colmeias, classificação dos possíveis predadores de abelhas e ameaças à colmeia. Caso ameaças sejam identificadas, um alerta será enviado para o apicultor e assim alertando sobre possiveis ataques à colmeia.

Objetivos

• Identificar possiveis ameaças à colmeia;
• Melhorar o tempo de ação do produtor em casos de ataques;
• Aplicar tecnologias de Inteligência Artificial no meio apicola.

↓ Aptação das imagens em tempo real da colmeia

↓ Transmissão das imagens captadas

↓ Extração das Informações por melo de redes neurais convolucionais

↓ Diferentes classificadores utilizam as extrações, e os melhores resultados são selecionados e então o sistema decide se faz ou não o envio de um alerta

• O produtor recebe um alerta para realizar a melhor tomada de decisão.

A implementação de Inteligência Artificial no meio apicola promete melhorias de produção, auxiliando na melhor tomada de decisão do apicultor.

- Projeto BEEWEB: Desenvolvimento de uma plataforma para monitoramento de dados de produção apícola.

O que é?

O projeto BeeWeb visa aprimorar o monitoramento de colmeias. O projeto utiliza sensores e uma plataforma web para coletar e apresentar informações cruciais, ofertando aos apicultores um modo eficiente de monitorar suas colmeias.

Objetivos

• Desenvolver um sistema de gerenciamento e monitoramento de estações apicolas.
• Desenvolver um sistema capaz de receber dados de um Arduino monitorando um apiário ou colmeia.

• Usuário cadastra apiário ou colmeia
Um usuário registrado é capaz de cadastrar seus apiários e colmeias integradas com Arduino no sistema.

• Apiário/colmeia envia dados para o sistema
Apiarios/colmeias monitorados por Arduinos enviamos dados periodicamente para o sistema online.

• Os dados são armazenados
Os dados enviados pelos arduinos são recebidos pelo sistema e armazenados no banco de dados. Os produtores poderão estimar o valor do animal e de sua carne com base nos gastos.

• Monitoração dos dados
Acessando o sistema os usuários podem visualizar os dados de seus apiários e/ou colmeias.

A aplicação que desenvolvemos representa um salto significativo na gestão e monitoramento de apiários e colmeias. Ao integrar a tecnologia do Arduino e a Internet das Coisas, estamos capacitando apicultores e entusiastas a acompanhar de perto o estado de suas colmeias de maneira mais eficiente e precisa.

- Estação Metereológica

O que é?

Uma estação meteorológica é um conjunto de instrumentos que coleta e mede dados climáticos, como temperatura, umidade, vento e chuva em um local especifico. Essas informações são cruciais para a previsão do tempo, agricultura, navegação e segurança.

Objetivos

• Coletar dados climáticos: Registrar informações sobre temperatura, umidade, pressão atmosférica e outros parâmetros para monitorar as condições climáticas locais.
• Previsão do tempo: Fornecer dados para melhorar a precisão das previsões meteorológicas, auxiliando na previsão de eventos climáticos futuros.

• Leitura dos Sensores
Coleta de dados com sensores DHT22, Velocidade de Vento, Pluviometro, UV e Direção do Vento.

• Armazenamento de dados

• Análise de dados

• Tomada de decisão

- Sistema de visão Computacional para caracterização do grau de maturação em bananas usando Redes Neurais Convulacionais

O que é?

O projeto possui um objetivo geral que exige outros específicos, dos quais um é a construção de um dataset especifico para atender aplicações condizentes com a fruticultura do semiárido cearense, especialmente na região centro-sul do estado.

Objetivos

• Desenvolver uma Visão Computacional para classificação do grau de maturação de bananas;
• Produzir um dataset de imagens de bananas produzidas na região centro-sul do Ceará;
• Ajustar um modelo de LA para o dominio da aplicação pretendida.

• Coleta de imagens para construção de um dataset que deverá ser utilizado para treinamento da Inteligência Artificial que fará o reconhecimento do grau de maturação das bananas.

• A construção do dataset envolveu um processo de rotulagem, onde 1000 imagens tiveram que ser classificadas manualmente, com auxilio de referências técnicas especializadas.

• O dataset foi publicado em evento científico da SBC e fica disponivel em repositório aberto na internet, para a comunidade cientifica interessada.

• O projeto encontra-se na fase de desenvolvimento de um modelo computacional para aprendizado de máquina que deverá classificar a maturação de bananas em um possível sistema de produção.

O trabalho objetiva um sistema automático, baseado em redes neurais convulacionais, para a caracterização do estado de maturação de bananas, a partir de imagens no pós-colheita. Com vista para o atendimento de uma possivel demanda do semiárido cearense, tornou-se necessário criar uma base de imagens de bananas em diferentes graus de maturação com amostra da fruticultura praticada na região centro-sul do Ceará. Apesar do rompimento de limites do cronograma, a produção desse dataset trouxe conquistas significativas, visto que proporcionou a publicação em evento cientifico com foco em Computação e Meio Ambiente. O trabalho segue em curso, no sentido de desenvolver um sistema de IA para aprender com o dotoset criado no projeto.
